Instructions: Press Start once to allow sounds on Safari. The timer will run 5 sets of 1:00 then 0:30 automatically. There is a 3-beep countdown before the first set and a 3-countdown before each sprint. "5 seconds to sprint" is announced at 0:05 of the 1:00 timer. A 5s countdown before "STEADY" at the end of the sprint is shown and spoken.
